# rust-pos - 中餐馆点餐系统 PRD

## 1. 产品概述

**项目名称**: rust-pos (中餐馆智能点餐系统)
**产品愿景**: 使用 Rust 技术栈构建的现代化中餐馆点餐系统，支持 Web + 桌面混合部署

## 2. 技术架构

### 2.1 技术栈

| 组件 | 技术 | 说明 |
|------|------|------|
| 前端 | Dioxus | React-like Rust UI 框架 |
| 后端 | Axum | Rust Web 框架 |
| 数据库 | SQLite (开发) / PostgreSQL (生产) | via SQLx |
| ORM | SQLx | 类型安全数据库操作 |
| 桌面 | Tauri | 桌面应用打包 |
| 异步 | Tokio | 异步运行时 |

## 3. 功能列表

### P0 (核心功能)

#### 3.1 点餐系统
- 顾客浏览菜单并选择菜品
- 购物车功能（增删改数量）
- 下单并选择支付方式
- 订单状态实时查看

#### 3.2 KDS (厨房显示系统)
- 订单队列实时显示
- 厨师标记菜品完成
- 菜品准备时间倒计时

#### 3.3 用户系统
- 用户注册/登录
- 用户信息管理
- 历史点餐记录

#### 3.4 支付系统
- 支持信用卡、Apple Pay、现金
- 支持小费（15%、18%、20%、自定义）
- 自动计算消费税

### P1 (重要功能)

#### 3.5 小票系统
- 客人小票
- 厨房小票

#### 3.6 第三方配送
- 配送订单接收

#### 3.7 离线模式
- 断网点餐

### P2 (扩展功能)
- 会员积分
- 优惠券
